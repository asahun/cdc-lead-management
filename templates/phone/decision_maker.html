Call Script – Decision Maker (CFO / AP / Controller)

Purpose: speak with the person who can authorize or oversee unclaimed-property review for [OwnerName]. Goal: permission to send details and open the door for DIY or full-service filing.

1. Opening

You:

Hi [ContactFirstName], this is [ProfileFirstName] with LoadRouter here in Georgia. I’m calling because we identified an unclaimed-property item reported under [OwnerName] that’s currently held by the State of Georgia (reference [PropertyID]).

(Pause and let them respond.)

2. Verification-First Statement

You:

This isn’t a solicitation call — the state publishes these records, and one of them matches your organization. I just want to send you the details so you can independently verify it on the state portal.

3. If They Want a Quick Summary

You:

It’s listed as a [PropertyType] from [HolderName], reported in [ReportYear] for about [PropertyAmount]. I can email you a short summary with the state reference so you can review it.

4. Ask for Permission to Send Details

Option A – Simple Ask

You:

Would it be okay if I email you the summary and the state’s verification link?

Option B – Mention DIY Option

You:

Some companies prefer to file it internally — I can include a brief direction sheet as well if that’s easier.

Option C – Mention Full-Service Option

You:

We also offer a full-service option on a success-based basis if you’d like us to handle preparation and submission, but there’s no obligation either way.

5. If They Ask About Cost

You:

There’s no fee unless funds are released to your organization. You’re always welcome to file directly with the state if you prefer.

6. Closing

You:

Great — I’ll send the verification summary shortly. If any questions come up after you review it, I’m always available. Thanks again for your time.
